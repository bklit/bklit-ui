title: Sidebar controls
description: Default shadcn Button and ToggleGroup in the Studio sidebar
---

Sidebar triggers use shadcn `Button variant="outline"`. Option sets use `StudioToggleGroup` (a layout wrapper over shadcn `ToggleGroup`) — see [Toggle groups](/docs/toggle-groups).

## Usage

```tsx
import { Button } from "@/ui/button";

<Button className="h-8 w-full justify-start text-xs" variant="outline" type="button">
  Chart type
</Button>
```
