`StudioToggleGroup` / `StudioToggleGroupItem` wrap shadcn `ToggleGroup` with `variant="sidebar"` and layout presets from `STUDIO_TOGGLE_LAYOUTS`. Use these for single-select option sets (2–7 choices), not `Tabs` — there is no tab panel.

**Source:** `packages/studio/src/components/controls/studio-toggle-group.tsx`

## Layouts

| Layout | Use case |
|--------|----------|
| `segmented` | Text segments (fill mode, ease/spring, export mode) |
| `icons` | Icon row (orientation, line cap, fade edges) |
| `cards-2` | 2-column preview cards (ease presets, curves) |
| `cards-3` | 3-column preview cards |
| `swatch` | Swatch + label (pie fill) |
| `legend` | 3×2 placement grid |

`IconToggleGroup` is a thin wrapper around `layout="icons"`.
