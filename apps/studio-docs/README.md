# Studio UI docs (local)

Fumadocs site for `@bklitui/studio` editor chrome — **not deployed**. Use for iterating on sidebar controls, motion UI, and pickers without the full Studio canvas.

```bash
# from repo root
pnpm --filter studio-docs dev
```

- Docs: http://localhost:3003/docs
- Search: `⌘K` / `Ctrl+K`
- Theme: header toggle or `D`

Content lives in `content/docs/`. Previews use shared demos from `packages/studio/src/dev/`.
