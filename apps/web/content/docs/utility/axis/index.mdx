---
title: Axis
description: X and Y axis components for line and area charts
---

Axis components add value and date labels to line and area charts.

## Components

- **[X Axis](/docs/utility/axis/x-axis)** — Date labels along the bottom (horizontal axis)
- **[Y Axis](/docs/utility/axis/y-axis)** — Value labels along the left (vertical axis)

Both components must be used inside a chart (`LineChart` or `AreaChart`) and render via portal into the chart container.
